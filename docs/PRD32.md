# PRD: "페이스 캔버스" (AI 얼굴형 진단 및 맞춤 스타일링 추천 사이트)

## 1. 프로젝트 개요
- **목표**: 사용자가 전면 사진을 업로드하거나 스마트폰 카메라로 촬영하면, AI가 얼굴형을 분석(계란형, 둥근형, 각진형, 긴형, 하트형 등)하고 그에 맞는 최적의 헤어스타일과 안경테를 추천해주는 뷰티 웹 서비스.
- **주요 목적**: **"얼굴형 진단" 관련 키워드 검색 결과 최상단 노출(SEO 1위)**, 트래픽 유입 극대화 및 **구글 애드센스(Google AdSense) 고단가 광고 수익화**.
- **핵심 가치**: 뷰티 스튜디오에 온 것 같은 세련된 UI 위에서, 나에게 딱 맞는 헤어와 안경 스타일링을 제안받고 친구들과 결과를 공유하는 경험 제공.

## 2. 구글 애드센스(AdSense) & SEO 한도 초과 최적화 전략
뷰티 및 패션 카테고리는 광고 단가(CPC)가 높은 편이므로, 양질의 텍스트가 생명입니다.
- **풍부한 결과 페이지 (텍스트 + 시각자료)**: "당신은 고급스럽고 지적인 매력을 가진 '각진 얼굴형(Square)' 입니다"와 같이 텍스트 매거진 형태로 결과를 길게 제공.
- **스타일링 칼럼 게시판 (블로그화)**: 애드센스 심사 통과 및 롱테일 키워드 유입을 위해 메인 기능 외에 "단발머리가 제일 잘 어울리는 얼굴형은?", "얼굴 작아 보이는 뿔테 안경 고르는 법" 등 2,000자 이상의 텍스트 아티클 10~20개 배치.
- **온페이지 SEO (검색어 최상단 노출 기법)**:
  - 핵심 키워드 매핑: `<title> 무료 AI 얼굴형 진단 테스트 - 내 얼굴형에 맞는 헤어/안경 추천</title>`
  - 구조화된 데이터(JSON-LD): `SoftwareApplication` (도구 제공) 및 `Article` (스타일링 팁) 스키마 추가.
  - 이미지 대체 텍스트(Alt text) 완비: 모든 추천 일러스트에 "둥근 얼굴형에 어울리는 레이어드 컷" 등 상세 묘사 삽입.
  - 모바일 최적화 및 로딩 속도 향상(Next.js ISR 활용).

## 3. 주요 기능
1. **AI 얼굴형 스캐너 (카메라/업로드)**:
   - 전면 카메라 캡처 또는 로컬 갤러리 이미지 업로드. (개인정보 보호를 위해 이미지는 서버에 저장하지 않고 브라우저단에서 바로 분석 후 폐기되도록 안내 문구 강조).
2. **진단 리포트 (Face Profile)**:
   - 가로/세로 비율, 턱선 각도, 광대 너비 등을 시각적인 스파이더 웹 차트나 수치로 보여줌 (뷰티 컨설팅 느낌 강화).
3. **헤어스타일 큐레이션**:
   - 성별 선택 시, 해당 얼굴형(예: 하트형)의 단점을 커버하고 장점을 살리는 헤어스타일 3가지 추천 (예: 사이드 뱅, 시스루 뱅, 허쉬 컷).
4. **찰떡 안경테 추천**:
   - 둥근 테, 사각 테, 캣아이, 반뿔테 등 얼굴 여백을 보완하는 최적의 안경 형태 추천.
   - *수익화 포인트*: 안경 추천 결과 하단에 안경 쇼핑몰 제휴 링크 연동(무신사, 젠틀몬스터, 일반 패션 안경테 등).

## 4. 활용 가능한 API 및 라이브러리 (비용 효율성 고려)
얼굴 세부 랜드마크(이목구비 포인트)를 잡아내어 비율을 계산하는 프론트엔드 솔루션이 가장 가성비가 좋습니다.

- **페이스 오피셜 라이브러리 (무료/브라우저 기반)**:
  - **MediaPipe Face Mesh (Google)**: 브라우저 환경에서 실시간으로 468개의 3D 얼굴 랜드마크를 추출하는 압도적인 1위 라이브러리. (가로 길이, 세로 길이, 턱선 각도 계산용 식 구축).
  - **face-api.js**: TensorFlow.js 기반으로 브라우저에서 얼굴 인식 및 랜드마크를 찾아주는 라이브러리. 이미지 분석에 탁월함.
- **설계 로직 (커스텀 구축)**:
  - MediaPipe로 점을 찍은 뒤, `(광대 폭 / 얼굴 전체 길이)` 및 `(턱선 각도)`를 수식으로 계산하여 조건부 로직 (if/else)으로 얼굴형 6가지(계란형, 둥근형, 각진형, 긴형, 하트형, 다이아몬드형)로 분류하는 자체 알고리즘 구현.
- **백엔드 AI API (대안, 유지비용 발생)**:
  - 고품질의 텍스트 결과를 생성할 필요가 있을 시 `OpenAI API (GPT-4o-mini)`에 얼굴형 측정 수치를 프롬프트로 넘겨 개인화된 뷰티 코멘트를 생성하게 함.

## 5. UI/UX 디자인 가이드 및 컬러 팔레트 (Beauty & Elegance)
- **컨셉**: 고급 화장품 브랜드 홈페이지나 프리미엄 뷰티 스튜디오 느낌. 사진이 돋보일 수 있는 뉴트럴하고 부드러운 스킨 톤과 차분한 악센트 컬러 사용.
- **Coolors 추천 팔레트**: [Elegant Nude & Rose Gold (클릭)](https://coolors.co/fdfbf7-f0e6dd-d3bba8-a77e70-4f3b43)
  - **Background (배경)**: `#FDFBF7` (크림 화이트 / 펄 화이트) - 전체적으로 화사하고 깨끗한 캔버스 느낌.
  - **Surface/Card (컨테이너)**: `#F0E6DD` (소프트 베이지) - 사진 업로드 영역이나 정보 카드 배경.
  - **Accent Muted (보조 포인트)**: `#D3BBA8` (베이지 우드 / 로즈골드 계열) - 구분선이나 태그, 부드러운 강조 요소.
  - **Accent Primary (주요 강조)**: `#A77E70` (더스티 로즈 / 차분한 벽돌색) - 중요 버튼, 활성화된 메뉴 아이콘, 결과 타이틀 등 메인 뷰티 포인트 컬러.
  - **Text (폰트)**: `#4F3B43` (다크 플럼 브라운) - 완전한 검은색이 아닌 딥 브라운/보라빛이 감도는 색으로, 눈에 피로를 주지 않고 우아한 느낌 부여.
- **추천 폰트 (Typography)**:
    - **타이틀/강조 (Serif)**: `Noto Serif KR` - 우아하고 고급스러운 매거진 느낌 강조. 브랜드의 신뢰도와 깊이감을 부여합니다.
    - **본문/UI (Sans-serif)**: `Pretendard` - 가독성이 뛰어나며 현대적인 Apple 스타일의 깔끔한 인상을 줍니다.
- **디자인 포인트**: 각진 사각형보다는 `border-radius: 20px` 이상의 둥글고 라운드 처리된 카드 디자인 적용. 얇고 우아한 세리프 폰트(타이틀)와 깔끔한 산세리프 폰트(본문)의 조화로운 매치를 통해 프리미엄 뷰티 스튜디오 컨셉 유지.

## 6. 개발 로드맵
- **Phase 1 (MVP - 핵심 기능 개발)**:
  - React/Next.js 기반 프론트엔드 구축.
  - `face-api.js` 또는 `MediaPipe` 연동 및 얼굴 랜드마크 추출 로직 구현 (업로드 기반 1차 테스트).
  - 6가지 얼굴형 분류 수식 로직 세팅 및 하드코딩된 헤어/안경 매칭 리스트 연결.
- **Phase 2 (UX 고도화 및 SEO 세팅)**:
  - 촬영 모드(웹캠 연동) 인터페이스 매끄럽게 다듬기 (원형 가이드라인 등 제공).
  - 뷰티 사이트 무드의 CSS 애니메이션 및 로딩 화면 구현 ("얼굴형 분석 중..." 스캐닝 효과).
  - 마크다운이나 MDX 구조로 얼굴형/스타일링 글이 발행되는 '뷰티 사전(블로그)' 라우팅 추가 (SEO용).
- **Phase 3 (광고 및 제휴 연동)**:
  - 구글 애드센스 결과 페이지 본문 내 광고 단위(In-article) 자연스러운 배치.
  - 결과 하단의 "추천 안경테 구매하기" 쇼핑몰 링크 등 Affiliate(제휴) 프로그램 연동.
